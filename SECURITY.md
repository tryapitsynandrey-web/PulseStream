# Security Policy

## Supported Versions

PulseStream is a public showcase portfolio. The following versions are actively monitored for security considerations:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of PulseStream very seriously. If you discover a vulnerability or insecure defaults inside this repository, please do not file a public issue.

Instead, please send a detailed email to: **security-reports@tryapitsynandrey.web** (or contact via portfolio links).

We will acknowledge your report within 48 hours and work with you to patch and coordinate a public release timeline.
